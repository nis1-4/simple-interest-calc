# Simple Interest Calculator

## Task 1

This repository contains the details and implementation of a **Simple Interest Calculator**.

The purpose of this project is to demonstrate how to calculate simple interest using a basic formula in programming.

## Formula Used

Simple Interest is calculated using the formula:

[
SI = \frac{P \times R \times T}{100}
]

Where:

* **P** = Principal amount
* **R** = Rate of interest
* **T** = Time (in years)

## Features

* Calculates simple interest based on user input
* Easy to understand and beginner-friendly
* Can be extended for more financial calculations

## Usage

1. Input the principal amount
2. Enter the rate of interest
3. Provide the time period
4. The program returns the calculated simple interest
